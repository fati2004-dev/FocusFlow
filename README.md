## 🚀 Overview

FocusFlow is a comprehensive productivity application that combines task management, focus tracking, document analysis, and an AI-powered coach to help users maximize their productivity and maintain momentum. The application features a modern, terminal-inspired UI with dark mode support and real-time analytics.

**Live Demo Features:**
- 📊 Track focus sessions and task completion
- 🤖 AI coach for motivation and guidance
- 📈 Visual analytics and progress tracking
- 📄 Document management and analysis
- ⏰ Pomodoro timer for focused work sessions

## ✨ Features

### Core Modules

| Module | Description | Status |
|--------|-------------|--------|
| **Dashboard** | Overview of key metrics and recent activity | ✅ Complete |
| **Tasks** | Create, manage, and track tasks with priorities | ✅ Complete |
| **Documents** | Upload and analyze documents for insights | ✅ Complete |
| **Focus Timer** | Pomodoro timer with session tracking | ✅ Complete |
| **Analytics** | Visual reports on productivity trends | ✅ Complete |
| **AI Coach** | Intelligent assistant for guidance | ✅ Complete |

### Key Functionality

- **Task Management**: Create, edit, delete, and complete tasks with priority levels (High/Medium/Low)
- **Focus Tracking**: Track focus sessions with automatic streak calculation
- **Productivity Analytics**: Visual charts for session trends and completion rates
- **AI Coach**: Contextual responses for planning, motivation, and exam preparation
- **Dark Mode**: Full dark/light theme support with persistent preference
- **Responsive Design**: Works on desktop and tablet devices
- **Local Storage**: Persistent data storage without backend dependency
- **Collapsible Sidebar**: Optimized screen space utilization
